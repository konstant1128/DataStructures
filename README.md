# Лабораторные работы

Этот репозиторий содержит лабораторные работы по программированию.

## Содержание

### Лабораторная работа 1 (`laba1.cpp`)
- **Язык:** C++
- **Описание:** Работа с остатками от деления
- **Примечание:** Сейчас код работает только с остатками, а не со строками, что делает его более производительным

### Лабораторная работа 2 (`laba2.cpp`)
- **Язык:** C++
- **Описание:** Умножение матриц с различными подходами к оптимизации
- **Производительность:**
  - Наивное умножение — 0.25 GFlops
  - Оптимизированный блочный алгоритм — 10.7 GFlops
  - BLAS dgemm — 42.3 GFlops
- **Тестирование:** Устройство с процессором AMD Ryzen 5 7520U, ОС Windows 11 Home

### Лабораторная работа 3 (`laba3.py`)
- **Язык:** Python

### Лабораторная работа 4 (`anagrams_lr4.py`)
- **Язык:** Python
- **Описание:** Поиск анаграмм

### Лабораторная работа 5 (`Google_problem_lr5.py`)
- **Язык:** Python
- **Описание:** Решение задачи от Google

## Как запустить

### Для C++ файлов:
```bash
g++ -o laba1 laba1.cpp
./laba1

g++ -o laba2 laba2.cpp
./laba2
```

### Для Python файлов:
```bash
python3 laba3.py
python3 anagrams_lr4.py
python3 Google_problem_lr5.py
```

## Требования

- Компилятор C++ (g++, clang++ или аналогичный)
- Интерпретатор Python 3.x
